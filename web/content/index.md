# GeoSpecimens Vocabulary Browser

A collection of SKOS controlled vocabularies for describing geological specimens, their materials, hazards, and constituent properties. Part of the GeoSpecimens Data Mobilization Platform.

## Browse

- [Vocabularies](/vocabs) — browse all concept schemes
- [Search](/search) — full-text search across every concept
- [Share](/share) — downloads, export formats, and embeddable components

| Vocabulary | Description |

| [Geological Specimen Material Type](/scheme?uri=http://vocab.geospecimens.org/def/geological-specimen-type) | Material types for geological specimens |
| [Geological Material Name Type](/scheme?uri=https://vocab.geospecimens.org/def/geological-material-name-type) | Naming conventions for geological materials |
| [Constituent Part Proportion](/scheme?uri=https://vocab.geospecimens.org/def/constituent-part-proportion) | Relative proportions of constituent parts |
| [Hazard Types](/scheme?uri=http://vocab.geospecimens.org/def/hazard-type) | Hazards associated with specimen handling |

## About the Platform

Every vocabulary is published as a static site with no runtime backend — concepts are available as machine-readable Turtle, JSON-LD, RDF/XML, JSON, and CSV, and can be embedded in any page via the `<prez-list>` web component.

Built on [SKOS](https://www.w3.org/2004/02/skos/), [DCAT](https://www.w3.org/TR/vocab-dcat-3/), and [SHACL](https://www.w3.org/TR/shacl/) using the [prez-lite](https://github.com/Kurrawong/prez-lite) publishing toolkit.
