Option E: Workspace Overview Page

The workspace/vocab selection page shows change status per vocab before you even open one. Each vocab card shows a mini change bar and expands to show layer details.

Concept: Before entering a vocab, the workspace page already tells you which vocabs have pending changes at each layer. This gives a bird's-eye view. Clicking a card expands it to show the layer breakdown. This complements any of the other options (A-D) used inside the scheme page.

Pros
  • See status of ALL vocabs before diving in
  • Change bar is instantly scannable
  • Helps prioritise which vocab to work on
  • Expandable detail keeps page clean
  • Complementary — works alongside any scheme-page option
  • Good for reviewers / managers checking status
Cons
  • Requires API calls per vocab to compute diffs (could be slow)
  • Workspace page becomes heavier / more complex
  • Only useful on workspace page, not during editing
  • Local unsaved changes can't be shown here (they're in-memory on scheme page)
